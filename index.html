/// Streaming agentic loop. Emits events over mpsc.
#[instrument(skip(self, tx))]
pub async fn run_with_events(
    &mut self,
    tx: Sender<AgentEvent>,
) -> Result<()> {
    loop {
        let response = self.client.stream_chat(&self.messages).await?;
        let tool_calls = response.tool_calls();
        if tool_calls.is_empty() {
            break;
        }
        for call in tool_calls {
            let approved = (self.approval_fn)(&call).await;
            if approved {
                let result = call.execute().await?;
                tx.send(AgentEvent::ToolResult(result)).await?;
            }

Features

Everything you expect.
Nothing you don't need.

Built from scratch in Rust. No Electron, no web views — just native egui rendering at 60fps.

Tree-sitter Highlighting

Semantic syntax coloring powered by real AST parsing, not regex. 24 token types mapped to theme colors. Works correctly inside strings, macros, and nested structures.

Multi-Tab Terminal

Full VTE terminal emulator with PTY support, 256-color rendering, and 10,000-line scrollback. Multiple sessions in tabs. Agent bash output streams directly into the terminal.

LSP Integration

rust-analyzer, pyright, typescript-language-server — auto-detected and spawned per workspace. Hover types, go-to-definition (F12), find references, autocomplete, and gutter diagnostics.

AI Modes

Five purpose-built modes: Chat, Ask, Debug, Plan, Edit. Each mode automatically collects the right context — file, selection, terminal output, or full project tree — before sending to the model.

Inline Diff Approval

When the agent edits a file, you see a full unified diff with color-coded hunks. Accept or reject each change individually before anything touches disk. Real control, not blind trust.

Works Offline

Ollama and LM Studio supported out of the box. Run any local GGUF model with no API key, no internet, no telemetry. Cloud providers (Claude, OpenAI, Groq) remain optional.
